# Security Research: AI Agent & Runtime Systems

I focus on engineering security risks in AI-enabled software delivery and automation environments.

## Current Research Areas

- AI-generated code vulnerability patterns in application and infrastructure code.
- Agent workflow security: tool invocation, shell execution, and trust-chain abuse.
- MCP security: connector trust boundaries, authorization models, and data exposure risks.
- AI runtime governance: permission minimization, telemetry, and incident response controls.
- AI DevSecOps integration across CI/CD, GitHub Actions, and cloud-native pipelines.
- Cloud and infrastructure security for autonomous and semi-autonomous developer workflows.

## Research Style

- Technical baselines and reproducible case studies.
- Security-first workflow modeling for production environments.
- Practical controls over policy theater.
