# Roadmap

This roadmap tracks planned research directions. Items are exploratory and may change as findings mature.

## Near-term

- Expand case studies for AI-assisted CI/CD and deployment workflows.
- Refine baseline controls with implementation examples.
- Add additional MCP security analysis notes for tool trust boundaries.

## Planned

- Build a permission-mapping method for agent-to-tool execution paths.
- Define test scenarios for workflow artifact trust and provenance failures.
- Draft practical detection heuristics for risky AI-generated workflow changes.
